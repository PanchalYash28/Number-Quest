# Number-Quest
A polished number guessing game . The project includes a live web version and a Java console version that follows the requirements.

## Features

- Random secret number generation
- Repeated guessing until win or attempt limit
- High, low, and correct feedback
- Safe invalid input handling
- Difficulty levels
- Attempt tracking, score, hint, replay, and guess history
- Static frontend ready for GitHub Pages, Netlify, or Vercel

## Run Locally

Open this folder in a terminal:

```bash
cd number-guessing-live-game
```

Start the local web server:

```bash
npm start
```

Then open:

```text
http://127.0.0.1:8765
```

You can also open `index.html` directly in a browser because this is a static HTML, CSS, and JavaScript project.

## Check The Code

```bash
npm run check:js
```

For the Java console version:

```bash
cd java-console
javac NumberGuessingGame.java
java NumberGuessingGame
```

## Definition Of Done Checklist

- Random number is generated correctly
- User can keep guessing
- High, low, and correct feedback works
- Invalid input is handled safely
- Optional features work without breaking the game
- Code is readable and ready to present
